# iPix

## Description

Photo management system for Intania.

## Installation

```bash
bun i
cd /apps/wen
bun i
npm i
```

## Running the app

```bash
bun dev
```

To run any particular app in the monorepo, use:

```bash
bun dev --filter <app-name>
```
